Introduction

Hi, my name is Muhammad Ashair Imran, welcome to my cloud-hosted resume. I am an aspiring Cloud Engineer and recent graduate in Computer Engineering from the University of Ottawa, as well as 4x certified in AWS. This website is a passion project I decided to create to refine and showcase my skills in AWS. The focus of this project is on the back-end AWS tools, hence the bare framework-less front-end shown to you now. Below I have listed all the features of the website and the various AWS tools I used. This project is on-going and will periodically be updated. Once the below features are implemented, other low-priority features will be added such as improvements to the front-end.

Features

Access and Credentials (Complete)

Access and credentials were setup in a professional way to prioritize security and industry standards.

  • Use of Root account avoided where possible
  • AWS Organizations used to manage multiple OUs
  • Different OUs for production and test.
  • AWS SSO (IAM Identity Center) from root to access accounts in OUs

This was done to simulate and gain experience in how professional cloud teams manage their accounts.

S3 Hosting (Complete)

This website is hosted on S3 with HTTPS for security and Amazon CloudFront for distribution. Amazon Route 53 to register domain.

Back-End API (In Development)

  • Database (DynamoDB)
  • API (API Gateway & Lambda)
  • Source Control, CI/CD

Front-end / Back-end Integration (To Be Developed)

  • Connecting the Back-end API to the front-end
  • Building automated tests
